ขั้นตอนการใช้งานโปรแกรมแปลงไฟล์ Convert .py to .exe 
(V 1.0 by Mr.Patchara Al-umaree)

**สิ่งที่ต้องมีก่อน
-Python3
-Pyinstaller (หากยังไม่มีโปรแกรมมีฟังก์ชั่นติดตั้งให้อัตโนมัติ)

1.รันโปรแกรม Program_py_to_exe.bat
(หาก PyInstaller ยังไม่ได้ติดตั้งในระบบ โปรแกรมจะแสดงเตือนให้ติดตั้ง PyInstaller
หากยอมรับติดตั้ง PyInstaller โปรแกรมจะทำการติดตั้งให้โดยอัตโนมัติ)
2.เลือกไฟล์ .py ที่ต้องการแปลง
3.เลือกไดเรกทอรีที่ต้องการเก็บไฟล์ .exe
4.(ถ้ามี) เลือกไฟล์ไอคอน .ico
5.เลือกตัวเลือกการสร้างไฟล์เดียว (--onefile) หรือการปิดหน้าต่างคอนโซล (--windowed)
6.กดปุ่ม Convert to EXE (ดูผลลัพธ์ใน ScrolledText)
7.กดปุ่ม Contact Developer เพื่อดูข้อมูลติดต่อผู้พัฒนา
8.ใช้ปุ่ม Reset เพื่อรีเซ็ตฟอร์ม (โปรแกรมจะบันทึกการตั้งค่าเมื่อปิดโปรแกรม)
**กรณีที่ต้องการติดตั้ง PyInstaller ในระบบ จำเป็นต้องมีสิทธิ์ในการดำเนินการติดตั้งแพ็กเกจใน Python (pip install)
